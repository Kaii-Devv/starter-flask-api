Dokumentasi API

Endpoint 1: Get Fresh Proxy

Deskripsi: Scrapt proxy socks gratis dari beberapa website dan Mengumpulkan proxy yang masih bisa di gunakan

Metode: GET

Endpoint: /api/proxy

Parameter:

Nama Tipe Contoh
type String socks4, socks5
timeout Integer default : 3 (detik)

Endpoint 2: Bing Image AI

Deskripsi: Membuat gambar menggunakan bing AI secara gratis

Metode: GET

Endpoint: /api/imagegen

Parameter:

Nama Tipe Contoh
prompt String girl+and+boy

Endpoint 3: Uptime

Deskripsi: Gunakan /api/uptime?target=https://web.com di cron job agar terhindar dari deteksi IP

Metode: GET

Endpoint: /api/uptime

Parameter:

Nama Tipe Contoh
target String https://web.com
